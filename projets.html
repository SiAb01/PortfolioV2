Chocolatein

Contexte : C’est un site qui répertorie l’ensembles des confiserie d’une boutique d’une chocolaterie . SGBDR : MariaDB , Langage : Front HTML/CSS - Javascript /Jquery Server Side : PHP Outils : Visual Code , Oracle VM VirtualBox , Azure Portail Ce site à été codé en respectant la structure mvc . Un déploiement de l’app en local dans un serveur Debian d’une machine virtuelle. Et également un deploiement de l’app web dans l’Azure Cloud enutilisant les l’App Service et Azure Database for MariaDB .
